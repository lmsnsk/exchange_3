s21_Smart_calc

The program is implemented on the QT graphics library. The calculator has the ability to calculate the result of the entered expressions including trigonometry and calculating the root of a number. It is also possible to display a graph of a function from X

The user can compose the entered expression. A graph will be built with default values where you you will be able to view the graph of the function despite its size, as there is a possibility of scaling.

Установка программы

Установка программы выполняется с помощью команды make install и производится в каталог SmartCalc на рабочий стол пользователя

Удаление программы

Удаление программы выполняется с помощью команды make uninstall

Архивация

Команда make dist позволяет произвести архивацию всей исходной конфигурации программы
