s21_Smart_calc

Программа разработана с использованием фреймворка Qt.

Калькулятор позволяет производить простые математическивы вычисления путем ввода выражения (сложение, вычитание, умножение, деление, возведение в степень, квадратный корень), а также вычичслять тригонометрические функции (cos, sin, tg, acos, asin, atg) и логарифмы (ln, log).

Так же возможна постройка графика функции по введенному выражение.

Помимо этого реализован режим кредитного калькулятора.

Установка программы

Установка программы выполняется с помощью команды make install и производится в каталог SmartCalc на рабочий стол пользователя.

Удаление программы

Удаление программы выполняется с помощью команды make uninstall.

Архивация

Команда make dist позволяет произвести архивацию всей исходной конфигурации программы.
